# Transaction-Tracker-Web
A website for tracking daily transactions, built using React.

Users can monitor all their:
- Bank transactions
- Credit Card transactions
- Stock investments  
- Mutual Fund investments  
- Miscellaneous transactions from other sources.


### Mobile App and API
- [Click here](https://github.com/ayushkumar-maurya/Transaction-Tracker) for the Transaction Tracker mobile app repository.
- [Click here](https://github.com/ayushkumar-maurya/Transaction-Tracker-API) for the API repository.


### Setup
- Install all the required packages using `npm i`.
- Open the `config.js` file located in the `src` directory and specify the API URL.
    ```
    const API_URL = ''
    ```
    Ex.:
    ```
    const API_URL = 'https://example.com/'
    ```
    **Note:** Remember to add a trailing forward slash (`/`) to the API URL.
- Start the website using `npm run dev`.
- Visit http://localhost:5173/ to view the website.
